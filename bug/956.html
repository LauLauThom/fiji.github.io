Description petersfbn 2014-11-07 09:58:54 CST 
I am trying to manually track particles in time lapse recordings to verify my automatic tracking. This works perfectly fine for one file (selecting spots and creating tracks in auto-linking mode). However, after saving the results of the first file I cannot process a second one - spots can be marked normally but aren't connected the way they should be. Instead every track is made up of a series of individual links and auto-linking mode can't be toggled anymore. This problem is resolved when restarting Fiji.

It seems to be linked to the spot editing tool to be registered twice on the second time the plugin is launched. So every key press results in having two spots created, and the linking mode cannot be toggled.
